### 题目一

说出`['1', '2', '3'].map(parseInt)`的执行结果

parseInt(解析的值, 解析值的进制) 然后返回10进制数

```js
parseInt('123', 5) // 将'123'看作5进制数进行解析，返回38
parseInt('2', 2) // '2', 不能转换为2进制所以返回NaN
parseInt('12', 2) // 1, 1可以转换为2进制返回1，'2', 不能转换为2进制所以忽略

// 题目解答如下
parseInt('1', 0) // 1
parseInt('2', 1) // NaN
parseInt('3', 2) // NaN
```
